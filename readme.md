

Spin up a gitpod workspace

[gitpod.io/#https://github.com/zvodd/gitpod-gcloud](https://gitpod.io/#https://github.com/zvodd/gitpod-gcloud)


Login

```
gcloud auth login
```

Set project id

```
gcloud config set project
```


Clone your repo

```
git clone <MY_APP_REPO> appfolder
cd appfolder
```

Deploy your project

```
gcloud app deploy
```
